# API reference

The public Python API of the `axiom` package, generated from the source
docstrings.

The two entry points most applications need are
{py:class}`axiom.session.Session` (the game loop) and
{py:class}`axiom.universe.Universe` (universe metadata); both are importable
directly from `axiom`. Everything else supports them.
